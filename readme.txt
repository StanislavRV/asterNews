Проект сайта новостей.

Front - React
Beack - Laravel
DB - mySql
Base name - aster

В React использованы:
 axios,
 moment,
 react-hook-form,
 sass,
 swiper


 После применения миграций и сидов будет создано 2 пользователя:
 admin - master@master.net  password'- master@master.net
 author - author@author.net  password'- author@author.net

Пользоветель может читать статьи, оставлять комментарии.
Автор создавать статьи и отправлять их на подтверждение публикации.
Админ публиковать, удалять статьи.
